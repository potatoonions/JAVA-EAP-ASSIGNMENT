API Reference

Base URL: /api — all endpoints return ApiResponse<T> JSON

Eligibility & Enrolment
GET/enrollment/eligibility/{studentId}?semester=

Check eligibility — returns CGPA, failed course count, and decision

POST/enrollment/{studentId}?semester=

Enrol student — advances level/semester if eligible

GET/enrollment/ineligible

List all ineligible student IDs

POST/enrollment/register

Register new student (userId, firstName, lastName, email, program, level, semester)

Academic Reports
GET/reports/semester?studentId=&semester=

Generate and persist semester report

GET/reports/yearly?studentId=&year=

Generate and persist yearly report

GET/reports/student/{studentId}

List all historical reports for a student

POST/reports/export?studentId=&semester=

Export report as plain text

POST/reports/email?studentId=&semester=&email=

Email report to recipient

User Management
POST/users

Create user account (userId, firstName, lastName, email, password, role)

GET/users/{userId}

Get user by ID

GET/users/email/{email}

Get user by email

GET/users

List all users

PUT/users/{userId}/status?status=INACTIVE

Update account status

POST/users/recover?email=

Recover inactive account

POST/users/{userId}/role?role=INSTRUCTOR

Assign role to user

Authentication
POST/auth/reset-password?email=

Initiate password reset

POST/auth/reset-password/complete?token=&newPassword=

Complete password reset

POST/auth/validate?email=&password=

Validate credentials

Notifications

Full email audit log

GET/notifications/audit/{email}

Audit log for one recipient

POST/notifications/scheduler/process

Manually trigger scheduler cycle
